GitLore Repo graph & chat

Sign in with GitHub using an OAuth App you control (PKCE). Put the Client ID in Settings; add the client secret there too if the token step fails (stored only in this browser).

Store builds can ship a pre-filled Client ID. Unpacked: open GitHub, create an OAuth app, paste the callback URL below (Copy), then save.

Chat needs a Gemini API key. Create a key at Google AI Studio, open , paste the key, then Save.
